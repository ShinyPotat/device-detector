# Device-Detector

## Instalación

```shell
git clone "https://github.com/ShinyPotat/device-detector.git"
cd device-detector
pip install -r requirements.txt
Modificar la interfaz de tu tarjeta de red en el código
Crear mac_addresses.json
```

## Formato del json

```json
{
    "dir_mac": "nombre",
    "hostname": "nombre"
}
```
